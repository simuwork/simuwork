# SimuWork

  <img src="docs/logo.png" alt="SimuWork logo" width="220" />
</p>

SimuWork is a lightweight project simulation built for Duke University's business class, giving students a safe sandbox to model workflows, test ideas fast, and demo product concepts without real-world risks.

## Highlights
- Build realistic project scenarios with adjustable constraints and team setups tailored for coursework
- Visualize outcomes instantly via the interactive `docs/index.html` dashboard
- Share simulations easily using the bundled demo and pitch assets in `docs/`

## Getting Started
1. Open `docs/index.html` to explore the interactive simulation experience.
2. Tweak project parameters, run the scenario, and iterate with your team.

Have fun experimenting!
